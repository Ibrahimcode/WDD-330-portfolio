Forms

The lagacy DOM have a usefull property called document.forms that returns a collection of all the form in the
document in the order they appear in the markup.

  1. form also use indexing.
    • eg. const form = document.forms[0];
  2. We can also search form using it name attributes
    • eg. const form = document.forms.search; (assume search is the form name.)
  3. Forms object also jave a method called elements that return an HTML collection of all the elemetns contain in the form.
    • eg. const [input, button] = forms.elements;
  4. We can also access forms controls using name attributes.
    • eg. const input = forms.searchInput;
    • eg. const input = forms[searchInput];
  5. forms.submit() will submit the form authomatically.
  6. forms.reset() will reset all the form controls back to it initial values.

Object Oriented Programming

Object Oriented Programming is a style of programming that involves separating the code into objects that
have properties and methods.

  1. The Main Concept of OOP

    1. Encapsulation
    2. Polymorphism
    3. Inheritance
  2. Classes are defined as a blueprint of an object; objects are then created as an instance of that class
    and inherited all the properties and methods of that class.
  3. Static Methods : are method that are called directly by it class rather than by instance of that class
  4. To make a property private, use an understand (_) as the first character in the property's name.
    • eg. _color
  5. A class can inherit from another class using the extentds keyword in the class declaration.

  6. Polymorphism:

    • The concept of polymorphism means that diffennt objects can have the same methods,
      implement them in the in different ways. This means the methods can be override.

Mixins:

Mixins is a way of adding properties and methods of some object to another object without using inheritance.

Modular JavaScript:

A module is a self contain peice of code that provide functions and methods that can then be used
in other file and by other modules. They allow you to keep part of a code in a self contained files.
