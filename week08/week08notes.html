Transforms

The CSS3 transform property lets you lets you translate, rotate, scale, and/or skew
any element on the page.

Translation

Translation elements can only be moved relative to it current position.

  1. If you wnat to move an element vertically or horizontally, your can use the
    'translate X' or 'translate Y' functions respectively.

  2. Scalling

    The scale(x,y) function scales an element by the define factors horizontally
    and vertically

  3. Rotation

    The rotate() function rotate an element around the the points of originby by a
    specifiend value.

  4. Skew

    The skew(x,y) function specifies a skew along the x and y axis.

Transitions

Transition allows the values of CSS properties to be change over time.

  1. Transition Property

    The transition-property property defines the CSS properties of the element that
    should be transitioned, with all for all properties being the default.

  2. Transition Duration Property

    The transition-duration property sets how long the transition will take: the
    duration of time it takes to go from the default state to the transitioned state.
    You can specify this either in seconds ( s ) or milliseconds ( ms ).

  3. Transition Timing Function

    The transition-timing-function lets you control the pace of the transition in
    even more granular detail.

  4. Transition Delay

    The transition-delay property, it’s possible to introduce a delay before the
    transition begins.

Multiple Transition

The transition properties allow for multiple transitions in one call.

If you want your properties to transition at different rates, or if you just
want a select few to have a transition effect, include them as a comma-separated
list containing, at minimum, the transition-property and transition-duration for each.

Animations

Keyframes

A keyframe is a snapshot that defines a starting or end point of any smooth transition.

To animate an element in CSS, you first create a named animation, then attach it
to an element in that element’s property declaration block.

Animation in themselves does not do anything, in other to animate an element,
you'll need to associate the animation with that element.

Drag And Drop

The drag and drop API allows us to specify that certain elements are draggable,
and then specify what should happen when these draggable elements are dragged
over or dropped onto other elements on the page.
