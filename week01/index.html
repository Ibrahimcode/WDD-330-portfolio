  1. Week01 notes

    1. Local storage is a native object use to store data in a key value pair. aside of cookies, localStorage can be use to maintain the state of an app. It can be use to store and retrieve data on storage. It can also remove data that it has previously write on storage

  1. Week01 code example

     
    1.                 
                    < !DOCTYPE html >
                    < html lang="en" >
                    < head >
                        < meta charset="UTF-8" >
                        < meta name="viewport" content="width=device-width, initial-scale=1.0" >
                        < title > First Code Example < /title >
                        
                        
      
                            < style >
                                textarea{
                            
                                    width:800px;
                                    
                                    height: 250px;
                                    
                                    background-color: lightyellow;
                                }
                            </style>
                        
      
                    < /head >
                    <body>
                            
      

                                < script >
                                    
                                    function loadStory(){
                                    
                                    var storyName = document.getElementById("name_input").value;
                                    
                                    var storyHTML = localStorage.getItem(storyName);
                                    
                                    document.getElementById("story_editor").value = storyHTML;
                                    
                                    }
                                    
                                    function saveStory(){
                                    
                                    var storyName = document.getElementById("name_input").value;
                                    
                                    var storyHTML = document.getElementById("story_editor").value;
                                    
                                    localStorage.setItem(storyName, storyHTML);
                                    
                                    }
                                    
                                    function displayStory(){
                                    
                                    var storyHTML = document.getElementById("name_input").value;
                                    
                                    document.getElementById("story_display").innerHTML = storyHTML;
                                    
                                    }
                                < /script >
                           
      
                        
                            
                            < h1 > Story Editor </h1 >
                            
                            Story Name: < input type="text" id=name_input></input>
                            
                            <section>
                            
                            <button onclick="loadStory()">Find and Edit It </button>
                            
                            <button onclick="saveStory()">Save It</button>
                            
                            <button onclick="displayStory()">Display It</button>
                            
                            </section>
                            
                            <textarea id="story_editor" placeholder="Write a totally awesome story
                            here."></textarea>
                            
                            <section id="story_display"></section>
                    </body>
                    </html>
