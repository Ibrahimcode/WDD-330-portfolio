"this" keyword

The value of "this" is the object "before dot", the one use to called the method.

  1. "this" is not bound.

    In JavaScript, the keyword "this" behaves unlike most other programming languages.
    It can be used in any function, even if it's not s method of an object.

  2. Calling "this" without an object will raise an error.
  3. In JavaScript, "this" is unbound or free, it value is evaluated at runtime as the object "before the dot."
  4. Arrow function have no "this", they rather take it from the outer context.
  5. Each property of an object is a "key-value pair".
  6. It is also possible to to create a new object using the "new object()" constructor.
  7. We can use dot(.) or [] square bracker notation to access an object property.
  8. We can also use dot (.) or [] square bracket notation to called a method.
  9. The 'in' operator can be used to check whether an object has a particular property.
  10. The 'hasOwnProperty()' can also be used to check if an object has a particular property.

Looping through an object.

We can loop through an object to accessd it properties.

  1. The 'Object.keys()', method returns an array of all the keys.
  2. Object.values(), returns an array of all the values.
  3. Object.entries(), returns an array of key-value pairs.
