Sets

A Set in JavaScript is a set of data structure that represent a collection of unique values.

  1. It cannot include any duplicated values.
  2. It is similar in context of a Mathematical set.
  3. It however, don't contain Mathematical set operations such as union, intersection, and product.
  4. A set is created usint the new operator and set() constructor:
    • const list = new set();
  5. You add values to a set using the add method:
    • list.add(1);

Memory leaks

Memory leaks occurs when a program retains reference to a value that can longer be accessed in its memory.

  1. Memory leaks can cause the averall sysmtem memory available to be reduce, and cause programs to run more slower.
  2. weakSet() can be used to prevent memory leaks by garbage collecting any reference to a "dead object".
